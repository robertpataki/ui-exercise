Social sharing

Share content with pop up intents.

Related files

  • javascripts/components/social-sharing-buttons.js
  • html/shared/components/social-sharing-buttons.html
  • stylesheets/components/social-sharing-buttons.scss
  • html/data/global.json

Config

The social sharing settings can be found in the global.json file, at the "social" entry.
The settings for Facebook and Twitter are used by the Twitter and Facebook meta partials to populate the appropriate meta tags in the head.

Code example

<a href="#" class="button fit-width vo-trigger" data-vo-id="Vpg9yizPP_g" data-vo-type="youtube">Play Youtube</a>
<script>const sharingButtons = new SocialSharingButtons();</script>

Window scrolling

Turn any anchor tag into a window scrolling trigger.

Related files

  • javascripts/components/window-scroller.js

Config

data-ws-el: CSS id or class of the element you want to scroll todata-ws-duration: Scroll durationUse 'ws-trigger' class to add any element to the trigger list.

Code example

<a href="#" class="button fit-width ws-trigger" data-ws-el="page">Scroll to top</a>
<script>const windowScroller = new WindowScroller();</script>
WindowScroller is a singleton.

Video overlay

Turn any anchor tag into a video overlay trigger.

Related files

  • javascripts/components/video-overlay.js
  • stylesheets/components/video-overlay.scss

Config

data-vo-id: The video id (either Youtube or Vimeo)data-vo-type: 'youtube' or 'vimeo'Use 'vo-trigger' class to add any element to the trigger list.

Code example

<a href="#" class="button fit-width vo-trigger" data-vo-id="Vpg9yizPP_g" data-vo-type="youtube">Play Youtube</a>
<script>const videoOverlay = new VideoOverlay();</script>
VideoOverlay is a singleton.

Video embed

Video embeds are fully responsive and always take up 100% of their parent's width. The videos are fitted to 16:9 ratio, but it can be changed from CSS.

Code example: Youtube (docs)

<iframe src="//youtube.com/embed/Vpg9yizPP_g?&rel=0&showinfo=0&loop=0&controls=1" frameborder="0" allowfullscreen></iframe>

Demo: Youtube

Code example: Vimeo (docs)

<iframe src="//youtube.com/embed/Vpg9yizPP_g?&rel=0&showinfo=0&loop=0&controls=1" frameborder="0" allowfullscreen></iframe>

Demo: Vimeo
